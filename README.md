# Blockgame

A software rendered 3D game about falling blocks, written in Go++, that uses SDL3.

![screenshot](img/screenshot.png)

Tested on Linux.

### Requires

* [Go++](https://github.com/vmaxer/gooop)

### Building

    make

### Running

    ./blockgame
